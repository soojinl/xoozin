
# Lotto Number Generator

## Overview

A simple web application to generate random lottery numbers.

## Project Outline

*   **`index.html`**: The main HTML structure of the application.
*   **`style.css`**: Styles for the application.
*   **`main.js`**: JavaScript logic for generating lottery numbers.

## Current Plan

1.  **Modify `index.html`**:
    *   Update the title to "Lotto Number Generator".
    *   Create a container to display the lottery numbers.
    *   Add a button to trigger the number generation.

2.  **Modify `style.css`**:
    *   Add styles for the overall layout, number container, and button to make it visually appealing.

3.  **Modify `main.js`**:
    *   Implement a function to generate 6 unique random numbers from 1 to 45.
    *   Add a click event listener to the button to generate and display the numbers.
